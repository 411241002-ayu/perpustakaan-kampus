# 📚 Aplikasi Sistem Informasi Perpustakaan Kampus

Aplikasi web perpustakaan sederhana berbasis PHP dan MySQL untuk memenuhi tugas mata kuliah Pemrograman Web. Aplikasi ini dirancang dengan antarmuka responsif dan modern bertema *Smokey Charcoal & Rose Magenta*.

## 🚀 Fitur Utama
* **Dashboard & Manajemen Buku:** Input data buku baru, edit detail buku, hapus buku, serta pencarian judul buku secara instan.
* **Kelola Anggota Peminjam:** CRUD (Create, Read, Update, Delete) data peminjam/anggota perpustakaan.
* **Sirkulasi Transaksi:** Pencatatan transaksi peminjaman buku dan pengembalian buku secara dinamis.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** HTML, CSS (Custom Styling & Glassmorphism), JavaScript.
* **Backend:** PHP Native.
* **Database:** MySQL / MariaDB.
* **Tools:** XAMPP, Visual Studio Code.

## 📦 Cara Instalasi & Penggunaan
1. Unduh atau clone repository ini.
2. Pindahkan folder project ke dalam direktori `C:/xampp/htdocs/`.
3. Buka XAMPP Control Panel, lalu jalankan module **Apache** dan **MySQL**.
4. Buka `localhost/phpmyadmin` di browser, buat database baru bernama `db_perpustakaan` (atau sesuaikan), lalu import file `.sql` database kamu.
5. Jalankan aplikasi di browser dengan mengakses `localhost/perpustakaan-kampus/`.
